# Tiny 16bit VM 

This is a little 16bit VM implementation which supports a few instructions such as:

- Push
- PopRegister
- AddStack
- AddRegister
- Mov

## Features
- 16-bit architecture
- Basic arithmetic and data movement instructions
- Simple stack-based operations
- Register-based operations

## How to execute

```sh
cargo run
```

## How to test
```sh
cargo test
```
